About StudentTrack

StudentTrack is a premium Student Management ERP built specifically for Diploma academic institutions to manage courses, marks transcripts, file storage, user profiles, and active analytics graphs.

Core Features

  • Secure Role Protection - Admins and users are completely separated with protected dashboard access.
  • SQLite Database Store - Securely tracks login sessions, upload paths, messages, and academic marks.
  • Student Activity Audit - Full activity logs tracking every log-in, file modification, and profile edits.
  • Transcripts & Downloads - Compiles student records into dynamically generated PDF reports.
  • Modern UI - Dark Theme, Bootstrap grid layout, glassmorphic styles, and interactive card hovers.

Technology Stack

Backend Framework Python 3 & Django 6.0
Database Store SQLite (relational, file-based)
Frontend Toolkit HTML5, Bootstrap 5, Custom theme.css
Visual Analytics Chart.js (responsive canvas metrics)
PDF Engine xhtml2pdf & reportlab
